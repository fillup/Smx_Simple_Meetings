# setAuthInfo( array $authInfo )

Set internal auth information. If authCode is not provided, one will be requested from the JoinMe API. The authCode can be reused and is valid until another one is requested, so you may persist it somewhere rather than the user's actual password. However, the JoinMe API does not have a concept of an application token as well, so if the user uses another service to access their account it will invalidate the authcode you have stored.
