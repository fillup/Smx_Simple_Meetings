Class Attendee

Attendee class provides methods for adding and listing attendees.

Attendee class extends the Site class in order to inherit common fields such as username, password, and sitename. Base class primariy provides getter/setter methods and each adapter that extends Attendee must implement actual functionality for interacing with APIs.
