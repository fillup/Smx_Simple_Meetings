Class Meeting

JoinMe Meetings class. Adds functionality for calling JoinMe APIs.

JoinMe is a very simple instant desktop sharing service and does not have many of the advanced features of other services. Therefore this adapter only implements the most basic ability to authenticate a user and create a meeting.
